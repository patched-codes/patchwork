# AgenticLLM Documentation

The `AgenticLLM` package is designed to manage conversational tasks with the use of a language model, integrating tools and strategies to effectively handle prompts and API interactions. The following documentation outlines the key components, input parameters, and expected outputs.

## Components

### 1. `typed.py`

This file contains type definitions for the inputs and outputs utilized by the `AgenticLLM` class.

#### Inputs

- **base_path (str):** The base directory where tools or other resources may be located.
- **prompt_value (Dict[str, Any]):** A dictionary containing key-value pairs that can be used to format or contextualize prompts.
- **system_prompt (str):** The prompt template that sets the context or tone for the conversation from the system's perspective.
- **user_prompt (str):** The template to guide user interactions or inputs to the model.
- **max_llm_calls (int):** Limits the maximum number of language model (LLM) calls. 
- **openai_api_key (str):** The API key for OpenAI, optional in context with other keys.
- **anthropic_api_key (str):** The API key for Anthropic, optional in context with other keys.
- **patched_api_key (str):** An alternative key required if specific conditions aren't met. Instructions for obtaining this key are provided in the code comments.
- **google_api_key (str):** The API key for Google services, optional in context with other keys.

#### Outputs

- **conversation_history (List[Dict]):** A list documenting the entire conversation history generated by the model.
- **tool_records (List[Dict]):** A list of records detailing tool usage and results during conversation execution.

### 2. `AgenticLLM.py`

This file defines the `AgenticLLM` class responsible for executing the agentic strategy using defined inputs to manage multi-turn conversations.

#### Usage

To create and run an instance of `AgenticLLM`:

```python
inputs = {
    "base_path": "/path/to/tools",
    "prompt_value": {"key": "value"},
    "system_prompt": "System prompt here...",
    "user_prompt": "User prompt here...",
    "max_llm_calls": 4,
    "openai_api_key": "your-openai-api-key"
}

agent = AgenticLLM(inputs)
result = agent.run()
print(result)
```

#### Core Operations

- **Initialization:** Sets up the agentic strategy with a limit on conversation turns.
- **Execution:** Manages the interaction with the LLM, utilizing defined tools, and records the conversation's history and tools used.

## Summary

The `AgenticLLM` package provides a structured way to interact with language models in a multi-turn conversation context, with configurable input options and clear output structures. By defining specific inputs such as API keys and prompt templates, users can tailor the conversation agent to suit their specific needs and manage interactions effectively through built-in tools and records.
